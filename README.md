# LibraryUiApp

This is the UI application for My Library Online project. It will contain all the user interface screens as well as frontend logic to lookup and display the resources. Code will be written in HTML and javascript using Angular framework. Wherever the data is needed from backend, it will make AJAX call to the respective API to fetch the data and display on screen. Additionally, the user update operations will also make call to API to make the database updates.
The application is intended to be used for college project.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.3.
